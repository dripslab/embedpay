# Contributing to embedpay

Thanks for helping improve embedpay. Contributions are welcome through GitHub pull requests.

## How to Submit a PR

1. Fork the repository and create a feature branch from `main`.
2. Keep your change focused on one fix or feature.
3. Add or update tests when behavior changes.
4. Run `npm test` before opening your PR.
5. Open a pull request against `main` with a clear title and description.
6. Link related issues with keywords such as `Closes #12`.

## PR Expectations

- Describe what changed and why.
- Include screenshots or short videos for UI changes.
- Keep generated files and unrelated formatting changes out of the PR.
- Be responsive to review comments and push follow-up commits to the same branch.

## Development

```bash
npm install
npm run dev
npm test
npm run build
```
